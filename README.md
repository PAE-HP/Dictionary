# Dictionary
 CANopen Object Dictionary definition for CANopenNode V4. This files were automatically generated by CANopenEditor v4.1-1-ga49a51a. We just have created a new personalized dictionary file using  CANopenEditor.


**Edición del diccionario CAN**


Tendremos dos diccionarios, uno para el master y otro para los slaves. Ambos se basan en la siguiente plantilla: https://github.com/CANopenNode/CANopenDemo/blob/master/demo/demoDevice.xdd 

**Cambios en común para ambos diccionarios:**

0x1005: Default value = 40000080	(activar syncs)

0x1006: Default value = 00002710      (10 ms)

0x1018: 0x01: Vendor- ID: Default value = FFFF0001

**Cambios master:**

0x1016: se añaden 24 sub índices más para tener 32 posibles slaves (8 default + 24)


**Cambios slave:**

0x1016: Índice eliminado
